quicklog-js
=========

JavaScript (ES6) client library (SDK) for the Quicklog.io API

## Installation

  `npm install quicklog`

## Usage

    var numFormatter = require('quicklog');

    var formattedNum = numFormatter(35666);


  Output should be `35,666`


## Tests

  `npm test`
